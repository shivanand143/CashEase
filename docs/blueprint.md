# **App Name**: CashEase

## Core Features:

- User Authentication: Enable users to securely register, log in, and manage their profiles with protected routes to ensure data privacy.
- Cashback Tracking: Allow users to browse stores and coupons, track clicks, and redirect to affiliate URLs while logging click timestamps.
- Cashback Management: Implement a system to fetch and match affiliate sales data with user clicks, manage cashback statuses (pending, confirmed, paid), and enable payout requests with admin approval.

## Style Guidelines:

- Primary color: Blue (#007BFF) for trust and reliability.
- Secondary color: Green (#28A745) for positive cashback associations.
- Accent color: Amber (#FFC107) for highlighting deals and CTAs.
- Clean and modern typography for readability.
- Use clear and intuitive icons for stores, coupons, and user actions.
- Responsive layout with a clear information hierarchy for easy navigation.
